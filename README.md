Array Extremes & Range Exclusion (C++)
Цей проєкт реалізує алгоритм аналізу масиву дробових чисел, який знаходить мінімальний та максимальний елементи та формує новий масив, виключаючи всі значення, що знаходяться між ними (включаючи самі екстремуми).

Технології:
C++.

Контейнер std::vector<double> — використаний для роботи з масивами дробових чисел довільної довжини.

Бібліотека <windows.h> — для коректної підтримки кириличного тексту в консольному інтерфейсі.

Що робить ця програма:
Пошук екстремумів: Програма за один прохід циклом знаходить індекси найменшого та найбільшого значень у масиві.

Динамічне визначення меж: Використовує функції min() та max() для визначення лівої та правої межі діапазону виключення, незалежно від того, в якому порядку зустрілися мінімум та максимум.

Фільтрація даних: Формує результуючий вектор, копіюючи лише ті елементи, що знаходяться поза межами знайденого діапазону.

Робота з дробовими числами: Використання типу double дозволяє обробляти точні обчислювальні дані.
